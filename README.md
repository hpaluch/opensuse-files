# openSUSE-files

Here are my configuration files and scripts for various openSUSE distributions.

* [leap16/](leap16/)
  1. install my addon packages (`audacious`, `keepass`, `virt-viewer` with SPICE) to LEAP 16.0
  2. install official ZFS packages - tested on LEAP 16.0
* [virt-manager/](virt-manager/) - override `virt-manager` defaults: disable scaling and enable auto resize
